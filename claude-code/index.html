前置準備

🛠️ 環境設置：安裝所有需要的工具

📥 第一步：安裝 Claude Code

Windows（PowerShell）： 

irm https://claude.ai/install.ps1 | iex

macOS / Linux： 

curl -fsSL https://claude.ai/install.sh | bash
  1. 安裝完成後，關掉 PowerShell / Terminal 再重新開啟一次，讓 PATH 生效。
  2. 輸入 claude --version，看到版本號就代表裝好了。
  3. 如果安裝或使用過程中遇到問題，可以參考 Claude Code 官方文件。

🟢 第二步：安裝 Node.js

前往 nodejs.org，點擊下方的安裝程式，下載後安裝即可。

🐍 第三步：安裝 Python

前往 python.org/downloads，點擊頁面上最新的 Python 3.14.x 安裝檔下載安裝。

安裝時記得把最下面的 Add python.exe to PATH 打勾，這樣 Claude Code 才能找到 Python。

🔑 第四步：申請 Tavily API Key 並裝上 MCP

Tavily 是給 AI 用的搜尋服務，免費帳號每月有 1,000 次搜尋額度，不用綁信用卡。裝上之後，Claude Code 才有比較好的網路搜尋能力。

申請 API Key

  1. 打開 app.tavily.com，點 Sign Up 註冊帳號（可以用 Google 帳號直接登入）。
  2. 登入後會看到你的 API Key。在 OPTIONS 中，點左邊第二個圖示就能複製 API Key。
  3. API Key 是你的帳號密碼，不要分享給別人，也不要貼在公開的地方。

裝上 MCP

貼上你的 API Key，下方指令會自動帶入。複製整段到終端機（PowerShell / Terminal）執行： 

claude mcp add tavily --scope user --env TAVILY_API_KEY=tvly-xxxxxxxxxx -- npx -y tavily-mcp@latest

看到 Added stdio MCP server tavily 就成功了。

新手上路

🖥️ 打開 Claude Code，認識你的工作環境

在資料夾裡開終端機、看懂對話介面、動手試一次就上手

📂 怎麼打開 Claude Code？

Claude Code 是命令列工具，必須在資料夾裡的終端機啟動。在電腦上新建一個空白資料夾（名字隨便取，但避免中文與空白）。

Windows：打開資料夾後，按住 Shift 同時點滑鼠右鍵，選 「在這裡開啟 PowerShell 視窗」。

macOS：打開 Terminal，輸入 cd 後把資料夾拖進去，按 Enter 切換到該資料夾。

確認終端機目前的位置是這個資料夾後，輸入： 

claude

畫面會切換成 Claude Code 的對話介面，最下面就是輸入框。

💡 每個專案用一個獨立的資料夾

Claude Code 會根據你啟動時所在資料夾的內容理解任務，如果把所有東西混在同一個資料夾，AI 容易搞混。

為了讓 AI 專注於當前任務，建議每個新專案都要開一個新資料夾進行維護。

🔐 第一次啟動會要求登入

Claude Code 會問你要怎麼登入。選 「Claude account with subscription」，瀏覽器會自動打開 claude.ai 登入頁面，用你訂閱 Pro / Max 的帳號登入即可。

看到 Login successful 就成功了。回到終端機會看到對話介面，輸入 /help 可以看所有可用的指令。

工具體驗 1/2

📋 體驗：做一份品牌分析報告

填三個欄位，把 prompt 丟給 Claude Code，它會用 Tavily 上網找資料、整理成 HTML

✏️ 填上你想分析的品牌 

請用 Tavily MCP 幫我針對「___」這個品牌做一份分析報告。

品牌資訊：
- 品牌名稱：___
- 主營品類：___
- 目前銷售國家：___

搜尋原則：每個主題最多搜尋 1-2 次就好，不用反覆追查，把時間留給分析。

請涵蓋以下三個主題，並針對搜到的資料做深入解讀：

1. 品牌概況：公司背景、主要產品線、品牌定位
2. Amazon 表現：熱賣產品、定價策略、評價中看到的優缺點
3. 近期動態與機會：近期新聞、市場趨勢，並給出你的觀察與建議

請著重在「分析與洞察」，不要只是把搜到的資料條列出來，多加入你的推論與串聯。
完成後把報告存成 report.html，用繁體中文撰寫，排版清楚好讀。

複製後貼到 Claude Code 輸入框，第一次用 Tavily 會跳權限確認，選 「Yes, and don't ask again for tavily」，之後就不會再問。完成後資料夾裡會多一份 report.html，雙擊打開看結果。

工具體驗 2/2

🧩 Skill：一鍵升級 AI 的專業能力

不用寫程式、不用設定，裝上 Skill 就能讓 AI 變成各領域的專家

🤔 Skill 是什麼？

簡單來說，Skill 就是寫給 AI 看的專案說明書，讓 AI 理解怎麼把一件事情做好，一種提煉於隱性智慧的顯性文檔。

🙌 安裝零門檻

不用寫程式、不用改設定檔，說一句話就能裝好。

⚡ 即裝即用

裝完直接生效，不需要累計或者沉澱更多使用記錄。

📐 可被監督

Skill 是純文字檔，可以藉由人工完成監察，判斷內容。

🔎 第一步：先裝一個會找 Skill 的 Skill

Skill 種類很多，與其自己去 GitHub 搜，不如裝一個 find-skills，之後想用什麼，請它去 Anthropic 官方 marketplace 找就好。把下面這段貼到 Claude Code 的輸入框： 

幫我安裝 Anthropic 官方的 find-skills plugin。
步驟：
1. 加入 Anthropic 官方 marketplace（如果還沒加的話）
2. 安裝 find-skills 這個 plugin
3. 安裝完 reload 一下，讓我可以直接在當前對話用

裝完後輸入 /，能在指令選單看到 find-skills 就成功了。

🎨 第二步：用 find-skills 找一個前端設計的 Skill

Anthropic 有一個 frontend-design 的 Skill，能讓 Claude 寫 HTML 時排版質感更好。但你不需要自己去找它的安裝指令，直接告訴 find-skills 你要什麼就好： 

用 find-skills 幫我找一個能讓前端設計變漂亮的 Skill，然後裝起來，
裝完後 reload，讓我可以在當前對話直接用。

📐 第三步：用 frontend-design 重做品牌報告

裝好之後，回頭把 Section 3 那份 report.html 重做一次。選好對象與風格，下方 prompt 會自動帶入，貼到輸入框送出： 

使用 frontend-design Skill，對 report.html 進行以下優化：

- 這份報告要給「___」看
- 設計風格：___
- 加入清楚的資訊層級：標題、小標、重點數據要一眼就能看到
- 數據部分用視覺化的方式呈現（例如卡片、標籤、色塊區分）
- 整體排版要專業、好讀，像一份正式的商業報告

直接修改 report.html，用繁體中文。

🔍 去哪裡找更多 Skill？

很多人會把自己做的 Skill 分享到 GitHub (開源社群，可以想像是工程師的 Thread) 上，直接去 GitHub 逛是最快的。

GitHub 上的星星數（⭐）就像點讚，星星越多代表越多人覺得好用，剛開始挑 Skill 的時候可以參考。

綜合實戰

🌐 實戰：用 AI 自動操作瀏覽器

讓 AI 幫你開網頁、點按鈕、抓資料，把重複的瀏覽器操作變成自動化流程

🤔 Claude for Chrome 是什麼？

Claude for Chrome 是 Anthropic 官方的 Chrome 擴充功能，可以讓 Claude Code 直接操作你平常在用的那個 Chrome 瀏覽器：開分頁、點按鈕、填表單、抓資料。

重點是它共用你 Chrome 的登入狀態，所以那些需要登入才看得到的網站（Amazon 賣家後台、Gmail、公司內網），它都能像你本人一樣進去操作。

🔐 共用登入

不用重新登入。你 Chrome 已經登入的網站，Claude 也可以直接進。

👀 看得到畫面

不是 headless 黑箱跑。整個操作過程在你眼前發生，可以隨時停下。

🔁 跨網頁工作流

在 A 網站抓資料、整理後丟到 B 網站，Claude 可以一條龍完成。

📦 安裝 Claude for Chrome 擴充功能

  1. 前往 Chrome Web Store 的 Claude 頁面，點 「加入 Chrome」 安裝擴充功能。
  2. 安裝後會出現在 Chrome 右上角，點開後用 同一個 Claude 帳號登入（就是你用 Claude Code 的那個帳號）。
  3. 回到終端機，關掉現在的 Claude Code（輸入 /exit），然後重新啟動時加上 --chrome 參數： 
claude --chrome

如果已經在 Claude Code 裡面，也可以直接打 /chrome 啟用連線。連上之後，下方狀態列會多一個 Chrome 圖示，代表 Claude 現在能看到你的瀏覽器了。

💡 只支援 Pro / Max / Team / Enterprise 訂閱

Claude for Chrome 目前是 Beta，需要付費訂閱帳號。Bedrock / Vertex AI 走 API 的學員暫時用不了。

🔍 實戰：去 Amazon 看一下你的品牌

Section 3 的報告是 Tavily 在公開網路上找到的資料。Amazon 站內的價格、評論細節，要實際打開頁面才看得到。這一步用 Claude for Chrome 把這部分補上。

Step 1：打開 Amazon 美國站、登入

在 Chrome 手動打開 amazon.com，用你的帳號登入。Claude 會用這個分頁的登入狀態操作。

Step 2：請 Claude 去抓品牌資料

品牌名稱會跟 Section 3 自動同步。下方 prompt 直接複製貼到輸入框： 

用我目前開著的 Chrome 幫我做以下事情：

1. 在 Amazon 搜尋「___」這個品牌
2. 從搜尋結果裡挑出 5 個熱門 ASIN（標題、售價、星級、評論數）
3. 點進其中評論最多的那一個產品頁，幫我抓最新 5 則評論（評分、標題、內文、日期）
4. 整理成一份 markdown 表格，存成 amazon-snapshot.md

過程中如果遇到驗證碼或登入要求，停下來告訴我，我會手動處理。

Claude 會在 Chrome 裡自己點頁面、抓資料，整個過程都看得到。如果它走錯方向，可以打斷它，告訴它「跳過這個」「再點下一頁」這類指令。

Step 3：把 Amazon 資料併回報告

把剛剛抓到的內容整合進原本的 report.html： 

把 amazon-snapshot.md 的內容整合進 report.html 的「Amazon 表現」段落，
重點放在：
- 熱門 ASIN 的價格帶與評論密度
- 評論裡反覆出現的優點和痛點
- 跟原本報告結論對比，有沒有需要修正的判斷

直接更新 report.html，保持 frontend-design Skill 的設計風格。
